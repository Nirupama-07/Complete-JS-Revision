## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/Complete-JS.git
```

Go to the project folder

```bash
cd Complete-JS
```

Install dependencies

```bash
npm install
```

Start Tailwind CSS

```bash
npx @tailwindcss/cli -i input.css -o output.css --watch
```
