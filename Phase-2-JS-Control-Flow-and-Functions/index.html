Javascript Control-Flow & Functions

Q(1)What is Control-Flow?

Answer:Control flow determines the order in which the javascript program should executes statements. Normally program runs from top to bottom but conditional statements, loops and functions can change the flow.

Q(2)What is Conditionals?

Answer:Conditional executes different blocks based on the specified condition which is either truthy or falsy.

There are 4 types of conditions: if , if-else , if...else-if...else , nested-if

if: Runs only when the condition is true

let age=parseInt(prompt("Enter your age"))
if(age>18){
console.log("You can drive")
}

if-else: It allows you to execute a block of code if the condition is true or another block of code will execute if the condition is false.

let age=parseInt(prompt("Enter your age"))
if(age>18){
console.log("You can drive")
}else{
console.log("You cannot drive")
}

if...else-if..else: It is used to check multiple condition in sequence and if one condition is true its block is executes and the rest are skipped.

let age=parseInt(prompt("Enter your age"))
if(age>18){
console.log("You can drive")
}else if(age<0){
console.log("Age cannot be negative")
}else{
console.log("You cannot drive")
}

nested-if: It allows to put an if statement inside another if statement.

let age=parseInt(prompt("Enter your age"))
if(age>20 && age<29){
console.log("You are in your 20's")
if(age%2==0){
console.log("Your age is even")
}else{
console.log("Your age is odd")
}
}else{
console.log("Your age does not lies between 20 and 29")
}

Q(3)What is switch?

Q(12)What is High-Order Function?

Answer:A function is a high order function if:

  • It passes one or more functions as an arguments
  • It returns a function

function applyOperation(x,y,operation){
return operation(x,y)
}
function add(a,b){
return a+b
}
console.log(applyOperation(9,6,add))

map, filter, and reduce are common higher-order functions.

const numbers = [1, 2, 3, 4];
const doubled = numbers.map(n => n * 2); // [2, 4, 6, 8]
const even = numbers.filter(n => n % 2 === 0); // [2, 4]
const total = numbers.reduce((sum, n) => sum + n, 0); // 10

Q(13)Explain all the popular HOF?

Answer:A high order function is a function which takes one or more functions as an argument and return a function.Some of the popular HOF are: map,filter ,reduce,forEach,find,some,every

map():The map function is used to transform an element by applying callback function to each elemnt. It returns a new Array.

e.g.
const n=[1,2,3,4,5]
const square=n.map((elem)=>elem*elem)
console.log(square)

filter():The filter function is used to create a new array containing element that satisfies the given condition. It returns a new Array.

e.g.
const n=[1,2,3,4,5]
const even=n.filter((num)=>num%2==0)
console.log(even)

reduce():The reduce function accumulates the array elements into a single value based on a callback function

e.g.
const n=[1,2,3,4,5]
const sum=n.reduce((acc,curr)=>acc+curr,0)
console.log(sum)
//acc->store the result of the value sum
//0->initial value

for-each():It executes a provided function for each array element. It does not returns a new Array.

e.g.
const n=[1,2,3,4,5]
n.forEach((num)=>console.log(num*2))

find():It returns the first element of the array which satisfies the given condition

e.g.
const n=[1,2,3,4,5]
const fEven=n.find((num)=>num%2==0) console.log(fEven)

some():It checks atleast one array element satisfies the condition. It returns the value in true or false

e.g.
const n=[1,2,3,-4,5]
const hasNeg=n.some((num)=>num<0)
console.log(hasNeg)

every():It checks that if every element in the array satisfies the condition. It returns value in true or false.

e.g.
const n=[1,2,3,4,5]
const hasNeg=n.some((num)=>num>0)
console.log(hasNeg)

Q(15)What is Scope?

Answer:Scope in JS is used to define where a variable can be access or used within a program. It can be of 4 types:Global Scope, Function Scope, Local Scope, Block Scope, Lexical Scope

Global Scope: Global scope refers to a variable which is declared outside any function or block and it can be accessed anywhere in the program either inside the function or outside

e.g.
const appName = "My App";
function showName() {
console.log(appName);
}

Local Scope:It is a variable which is declared inside a function or block and cannot be accessed outside that function.

e.g.
function showName() {
const appName = "My App";
console.log(appName);
}

Function Scope:It is similar to local scope, a variable which is declared inside a function by using let var or const and cannot be accessed outside that function.

e.g.
function showName() {
const appName = "My App";
console.log(appName);
}

Block Scope:It is similar to local scope, a variable which is declared inside a { }by using let or const and cannot be accessed outside that block.

e.g.
function showName() {
const appName = "My App";
console.log(appName);
}

var is not a block scoped

Lexical scope:It means a function can access variables from the scope where the function was defined—not where it is called.

e.g.
const outer = "outside";
function parent() {
const inner = "inside";
function child() {
console.log(outer); // outside
console.log(inner); // inside
}
child();
}
parent();

Q(18)What is closure?

Answer:A closure is created when an variable of an inner function can access the value from an outer function even if the outer function has finished executing

function createCounter() {
let count = 0;
return function () {
count++;
return count;
};
}
const counter = createCounter();
console.log(counter()); // 1
console.log(counter()); // 2
console.log(counter()); // 3

Uses:

  • Data privacy and encapsulation
  • Counters
  • Event Handlers and callbacks
  • Memorization
