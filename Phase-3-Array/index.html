Javascript Array

Q(1)What is an Array?

Answer:An array is an ordered lists of values. Each values known as an element, is assigned to a numeric position called as index. Array can hold any type of data such as numeric, string,objects or even other arrays making them flexible.

e.g. const fruits=["apple","mango","cherry"]

e.g. Empty array: let arr=[ ]

Q(2)How can you create an array?

Answer:There are 2 ways to create an array.

Array Literal: let arr=["Mango",23,54.6,true]

Array Constructor: let arr= new Array("Mango",23,54.6,true)(Avoid unless neccessary)

Q(3)List all the important properties to perform on array.

Answer:The important properties to perform on array are:

length: Returns total number of elements.

e.g. let arr=["Aman","Sudipta","Baishnovi"]
console.log(arr.length) //output: 3

Array Methods

push: Adds element at the end of the array.

e.g. let arr=["Aman","Sudipta","Baishnovi"]
arr.push("Arav")
console.log(arr) //output:["Aman","Sudipta","Baishnovi","Arav"]

pop: Remove element at the end of the array.

e.g. let arr=["Aman","Sudipta","Baishnovi"]
arr.pop()
console.log(arr) //output:["Aman","Sudipta","Baishnovi"]

shift: Remove element at the start of the array.

e.g. let arr=["Aman","Sudipta","Baishnovi"]
arr.shift()
console.log(arr) //output:["Sudipta","Baishnovi"]

unshift: Add element at the start of the array.

e.g. let arr=["Sudipta","Baishnovi"]
arr.unshift("Aman")
console.log(arr) //output:["Aman","Sudipta","Baishnovi"]

splice: Remove or replaces or add the element from the array.

syntax:array.splice(start, deleteCount, items...)
let arr=["Aman","Sudipta","Baishnovi"]

Remove:

arr.splice(1,1)
console.log(arr) //output:["Aman","Baishnovi"]

Add:

arr.splice(1,0,"Simran")
console.log(arr) //output:["Aman","Simran","Baishnovi"]

Replace:

arr.splice(2,1,"Rahul")
console.log(arr) //output:["Aman","Simran","Rahul"]

slice: Returns a new array without modifying the original.

e.g. let arr=["Aman","Simran","Rahul","Vaishnavi","pankaj"]
let result=arr.slice(1,4)
console.log(result) //output:["Simran","Rahul","Vaishnavi"]

concat: Used to combines arrays.

e.g. let arr1=["Aman","Simran","Rahul","Vaishnavi","pankaj"]
let arr2=["1","2","3","4","5"]
let result=arr1.concat(arr2)
console.log(result) //output:["Aman","Simran","Rahul","Vaishnavi","pankaj","1","2","3","4","5"]

includes: Check if element exists.

e.g. let arr1=["Aman","Simran","Rahul","Vaishnavi","pankaj"]
console.log(arr1.includes('2')) //output: true

indexOf: Returns the first index.

e.g. let arr1=["Aman","Simran","Rahul","Vaishnavi","pankaj"]
console.log(arr1.indexOf("Rahul")) //output: 2

if you want to access an index of an element which is not present then it will return -1

find: Returns the first matching value.

e.g. let arr1=['1','2','3','4','5']
console.log(arr1.find(num=>num>2)) //output: 3

findIndex: Returns index of the first matching element.

e.g. let arr1=['1','2','3','4','5']
console.log(arr1.findIndex(num=>num>2)) //output: 2

some: Returns true if only one element satisfies the condition.

e.g. let arr1=['1','2','3','4','5']
console.log(arr1.some(num=>num>4)) //output: true

every: Returns true if all the element satisfies the condition.

e.g. let arr1=['1','2','3','4','5']
console.log(arr1.every(num=>num>4)) //output: false

sort: Returns the sorting array.

e.g. let arr1=['3','5','1','2','4']
console.log(arr1.sort()) //output:['1','2','3','4','5']

For number always use .sort((a,b)=>a-b) for ascending

reverse: Reverse an array.

e.g. let arr1=['1','2','3','4','5']
console.log(arr1.reverse) //output: ['5','4','3','2','1']

join: Converts an array into a string.

e.g. let arr1=['1','2','3','4','5']
console.log(arr1.join("-")) //output: [5-4-3-2-1]

flat: Flattens nested array.

e.g. let arr1=['1',[2,'3',4],'5']
console.log(arr1.flat()) //output: ['1',2,'3',4,'5']

flatMap: Map each element and then flattens the result by each value.

e.g. let arr1=[2,3,5]
console.log(arr1.flatMap((num)=>num*2)) //output: [4,6,10]

Array.isArray: Checks if a value is an array.

e.g. console.log(Array.isArray(['1','2'])) //output: true

Array.from(): Creates an array from an iterable or array like object.

e.g. console.log(Array.from("Hello")) //output: ['H','e','l','l','o']

Array.from(): Creates an array from a given argument.

e.g. console.log(Array.of(1,2,3)) //output: [1,2,3]

Q(4)How can you iterate over an array?

Answer:Array Iteration Methods

forEach(): Loops through an array.

e.g. let arr=["aman","sangeet","rani"]
arr.forEach((ch)=>{
console.log(ch)
})
//output:
aman
sangeet
rani

for..of:A clean way to iterate over values

e.g. let arr=["aman","sangeet","rani"]
for(const ch of arr){
console.log(ch)
}
//output:
aman
sangeet
rani

map(): It transforms each element in an array and returns a new array.

e.g. let arr8=[20,40,30] console.log(arr8.map(num=>num*2))
//output:[40,80,60]

filter(): Filters the array and returns a new array of matching elements.

e.g. let arr8=[20,40,30] console.log(arr8.filter(num=>num>30))
//output:[40]

reduce():Reduces an array into one element.

syntax:array.reduce((accumulator,current)=>{},initialValue) let arr8=[20,40,30] let sum=arr8.reduce((acc,num)=>{return acc+num},0)
console.log(sum)
//output:90

Q(5)What is Destructuring?

Answer:Destructuring is a JavaScript feature that allows you to extract values from arrays or properties from objects into separate variables.

Array Destructuring: const fruits = ["Apple", "Mango", "Banana"];
const [first, second, third] = fruits;
console.log(first); //Apple
console.log(second); //Mango
console.log(third); //Banana

Object Destructuring:
const student = {
name: "John",
age: 22,
city: "Delhi"
};
const {name, age} = student;
console.log(name); //John
console.log(age); //22

Q(6)What is Spread Operator(...)?

Answer:The spread operator (...) expands an iterable (like an array, string, or object) into individual elements or properties.

Copy Array: const arr1 = [1, 2, 3];
const arr2 = [...arr1];
console.log(arr2);

Merge Array:
const arr1 = [1, 2, 3];
const arr2=[4,5,6]
const arr3 = [...arr1,...arr2];
console.log(arr3);

Add Elements:
const arr1 = [1, 2, 3];
const arr2 = [7,4,...arr1,9];
console.log(arr2);

Update objects:
const user = {
name: "John",
age: 20
};
const updated = {
...user,
age: 25
};
console.log(updated);

Q(7)What is Rest Operator(...)?

Answer:The rest operator (...) collects multiple elements into a single array or object.Although it uses the same ... syntax as spread, its purpose is different.

Rest in Array Destructuring:
const arr = [1, 2, 3, 4, 5];
const [first, ...rest] = arr;
console.log(first); //output: 1
console.log(rest); //output:[2,3,4,5]

Rest in object Destructuring:
const student = {
name: "John",
age: 20,
city: "Delhi"
};
const {name, ...others} = student;
console.log(name); //output: John
console.log(others); //{age:20,city:"Delhi"}
