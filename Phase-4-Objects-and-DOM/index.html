Javascript Objects and DOM

Q(1)What is object?

Answer:An object in javascript is a collection of related data in key-value pairs where each key uniquely identifies its values.

e.g.
const student={
  name:"Nirupama",
  age:23
}
console.log(student)

e.g.

}
console.log(student)

const student2=new Object()
student2.name="Priti"
student2.age=23

Q(10)What is this keyword in object methods?

Answer:In javascript object methods, this usually refers to the object that called the method.

e.g.

  }

profile.greet()

Q(13)What is object destructuring?

Answer:Object destructuring lets you pull properties from an object into separate variables.

e.g.
const person = {
  name: "Asha",
  age: 25
};
const { name, age } = person;
console.log(name); // "Asha"
console.log(age); // 25

Q(15)What is differnce between shallow copy and deep copy?

Answer:

Shallow copy:A shallow copy copies only the first level of an object. If the object contains nested objects or arrays, those nested values are shared between the original and the copy.

e.g.
const person = {   name: "Rahul",
  address: {
    city: "Delhi"
}
};
const copy = { ...person };
copy.name = "Amit";
  console.log(person.name);
  console.log(copy.name);

Deep copy:A deep copy creates completely new copies of the object and all of its nested objects or arrays.

e.g.
const person = {
  name: "Rahul",
  address: {
 : city: "Delhi"
  }
};
const copy = structuredClone(person);
copy.address.city = "Mumbai";
console.log(person.address.city);
