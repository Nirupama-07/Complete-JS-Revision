Javascript Objects and DOM

Q(1)What is DOM?

Answer:The JS HTML DOM reprsents the structure of an HTML document as a tree objects.It allows JS to interact with and modify the content and elements of a webpage dynamically without reloading it.

Features:

  • Tree structures:Represent the HTML document as a hierarchical tree of nodes.
  • Element access:Provides methods to select element such as getElementById(),getElementByTagName() and querySelector().
  • Dynamic Manipulation:Allows changing content, attributes and styles dynamically.
  • Event Handling:Support handling users interactions like click,doubleclick,mouse and keyboard events

Q(2)How can you select element in DOM?

Answer:There are five ways to select element in a DOM.

  • .getElementById()
  • .getElementByClassName()
  • .getElementByTagName()
  • .querySelector():Returns the first matching element.
  • .querySelectorAll():Returns all matching elements.

Q(3)How can you change a content in DOM?

Answer:There are two ways to change a content in a DOM.

  • .textContent():Reads only text
  • .innerHTML():Reads html

Q(4)How to change CSS in DOM?
