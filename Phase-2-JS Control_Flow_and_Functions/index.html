Javascript Control-Flow & Functions

Q(1)What is Control-Flow?

Answer:Control flow determines the order in which the javascript program should executes statements. Normally program runs from top to bottom but conditional statements, loops and functions can change the flow.

Q(2)What is Conditionals?

Answer:Conditional executes different blocks based on the specified condition which is either truthy or falsy.

There are 4 types of conditions: if , if-else , if...else-if...else , nested-if

if: Runs only when the condition is true

let age=parseInt(prompt("Enter your age"))
if(age>18){
console.log("You can drive")
}

if-else: It allows you to execute a block of code if the condition is true or another block of code will execute if the condition is false.

let age=parseInt(prompt("Enter your age"))
if(age>18){
console.log("You can drive")
}else{
console.log("You cannot drive")
}

if...else-if..else: It is used to check multiple condition in sequence and if one condition is true its block is executes and the rest are skipped.

let age=parseInt(prompt("Enter your age"))
if(age>18){
console.log("You can drive")
}else if(age<0){
console.log("Age cannot be negative")
}else{
console.log("You cannot drive")
}

nested-if: It allows to put an if statement inside another if statement.

let age=parseInt(prompt("Enter your age"))
if(age>20 && age<29){
console.log("You are in your 20's")
if(age%2==0){
console.log("Your age is even")
}else{
console.log("Your age is odd")
}
}else{
console.log("Your age does not lies between 20 and 29")
}

Q(3)What is switch?

Answer:The switch-case statement is used for decision making purposes. In some cases it is more convenient than if-else-if.

const day = "Monday";
switch (day) {
case "Monday":
console.log("Start of week");
break;
case "Friday":
console.log("Almost weekend");
break;
default:
console.log("Regular day");
}

Answer:Loop is used to execute a block of code repeatedly. It is of following types: for loop , while loop , do-while loop

for loop: It executes a block of code until the specified condition is met.

while loop: It executes a block of code the condition is true.

let i = 1;
while (i <= 5) {
console.log(i);
i++;
}

do-while loop: It executes a block of code atleast once even if the condition is false.

let value = 10;
do {
console.log("Runs once");
} while (value < 5);

Q(6)What is break and continue?

Answer:break:it is used to terminate or stops the code when the condition is met.

for (let i = 1; i <= 10; i++) {
if (i === 5) {
break;
}
console.log(i);
}

continue:it is used to skips the current iteration and moves to the next iteration.

for (let i = 1; i <= 10; i++) {
if (i === 5) {
continue;
}
console.log(i);
}

Q(7)What is Nested Loops?

Answer:It is a loop inside another loop. The inner function will complete all its iteration for each iteration of outer loop.

for (let row = 1; row <= 3; row++) {
for (let col = 1; col <= 2; col++) {
console.log(`Row: ${row}, Column: ${col}`);
}
}
