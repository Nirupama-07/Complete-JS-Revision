Javascript Fundamentals

Q(1)What is Javascript?

Answer: Javascript is a high level programming language,JIT compiled language that adds logic and functionality to the web. It turns static pages into apps-handling clicks, form,data fetching and real-time updates. It is dynamically typed,prototype-based and support event-driven and asynchronous programming.It can run in the browser and can also run in the server by using environments like NodeJS.

Q(2)What are the features of JavaScript

Answer: Features of Javascript are:

  • Lightweight and can easily be embedded in HTML
  • Runs both on client side and server side
  • Dynamically typed which means variables types are decided during runtime
  • Supports object oriented programming by using prototype and classes
  • Event driven which helps in performing events such as click on button, mouse events,keyboard events
  • Performs asynchronous tasks like promise,callback and async & await
  • Has First class functions:functions can be stored as a variable name,can be passed as an argument, can be used to call inside another function.

Q(3)Is JS compiled or interpreted?

Answer: Javascript is mostly interpreted but modern browsers like chrome's V8 engine compile it just-in-time(JIT) to make it faster. JS first reads your code line-by-line and runs it directly. Modern browsers like chrome's V8 engine first translate parts of your code and convert it into machine code while running it, so the computer can execute it much faster.

Q(4)What are variables?

Answer: Variables are thec ontainer that stores data values so that it can be accessed and modified later.
There are 3 ways to declare a variable:

  • var:Declares variables within the function or globalscope and allows re-delaration and updates within the same scope.These are hoisted and initialized with undefined.
  • let:Declares variable within the block scope, alllowing updates but not re-declaration within the same block.
  • const:Declares variable within the global scope that cannot be reassigned after their initial assignment.

Q(6)What is Type Conversion and Type Coercion?

Answer: Type Conversion:

Type Conversion is the explicit conversion of a datatype to another datatype.e.g.

  • Number('123')
  • String(123)
  • boolean(1)
  • parseInt("123")

Type Coercion:

In type coercion JS automatically converts a datatype to another datatype. e.g.

  • '5' + 2=52
  • '5' - 2=3
  • true + 1=2

Notes:5 == "5"; // true — loose equality allows coercion 5 === "5"; // false — strict equality checks type and value

Q(7)What are operators?

Answer: Operators are the symbols that perform some operations on the values.

  • Arithmetic Operator: + , - , * , / , %
  • Assignment Operator: += , -= , *= , /= , **=
  • Comparision Operator: == , === , != , !== ,>= , <=
  • Logical Operator: && , || ,!
  • Bitwise Opearator: & , | , ^ , ~
  • Ternary Operator: condition ? true : false

Q(8)What are Template literals?

Answer: Template literals follows backticks(` `) and allows interpolation and multi-line strings.

console.log( `Hello my name is: ${name} ` )
